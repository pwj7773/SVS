# 숙박업소 예약 데이터 시각화 프로젝트

### Smart Vision Service팀


- 프로젝트 목표
  예약업무 자동화 서비스 '김실장'을 통해 추출된 예약데이터를 기반으로 시각화된
  예약 데이터를 이용자에게 제공 할 수 있으며, 이를 통해 영업사원의 영업활동 및
  숙박업소의 객실 판매전략 수립에 도움을 줄 수 있는 웹 사이트를 개발을 목표로 한다.
